## XOXO
A clojure based tic tac toe game.

### Features
* A two player game.
* Contains tally board to view the results.
* Players can play indefinitely.

### Usage

*   Clone the code
*   Ensure lein works
*   Run `lein uberjar`
*   To start the game, run 
    ```
    java -jar target/uberjar/tic-tac-toe-game-standalone.jar
    ```
